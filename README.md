# Créer un convertisseur de chiffres

## romains vers arabes

ex:
const nine = romanToNumber("IX")
Tu as le droit d'avoir un objet qui ressemble à ça au départ, pour faire tes correspondances:
const romanNumbers = {
 "I": 1,
 "V": 5,
 "X": 10,
 "L": 50,
 "C": 100,
 "D": 500,
 "M": 1000
}

## Puis l'inverse

const romanNine = numberToRoman("9")
